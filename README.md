# python_study_myself

## 简介
本仓库用于记录我在 **Python 编程、算法实现、机器学习理论与工程实践** 方面的学习过程。  
目标是系统掌握 Python 技术栈、常用算法原理与代码实现，并积累完整的学习路径。


## 学习方向总览
- Python 基础 语法、函数、OOP、模块化、异常、文件IO
- 数据结构与算法分析  
- Python 工程开发  包结构、虚拟环境、Git、VSCode、命令行工具 
- 数据分析与可视化  NumPy、Pandas、Matplotlib
- 机器学习基础 监督学习、非监督学习、评估指标 
- 常用算法实现  KNN、SVM、Decision Tree、ID3、KMeans、PCA、LDA 
- 工程实践 项目部署、模型保存、可视化、优化
